# Calculadora Inteligente

Projeto desenvolvido em Python com Flask, HTML, CSS e JavaScript.

A proposta inicial era criar uma calculadora simples em Python, mas o projeto foi evoluído com uma interface web para melhorar a visualização e a experiência do usuário.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de erros
- Validação de campos
- Interface web responsiva

## Tecnologias utilizadas

- Python
- Flask
- HTML
- CSS
- JavaScript

## Objetivo do projeto

Praticar lógica de programação usando entrada de dados, conversão de tipos, condicionais, funções, tratamento de erros e estruturação básica de uma aplicação web.
